# portalcn-leyu-lab

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态网页形式存在，不依赖于特定域名或网站架构，适合用于展示、测试、存档或轻量级应用场景。

## 目录说明

```
/
├── index.html          # 仓库页面索引（可选）
├── pages/              # 存放各独立 HTML 页面
│   ├── page1.html
│   ├── page2.html
│   └── ...
├── assets/             # 公共资源（样式、脚本、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `pages/` 目录下存放各个独立的 HTML 页面文件，每个文件可独立运行。
- `assets/` 目录存放页面共享的样式、脚本、图片等资源文件。

## 页面归档说明

- 每个 HTML 页面均为独立文件，可直接在浏览器中打开运行。
- 页面之间无强依赖关系，可以单独复制或部署。
- 归档内容可能来自不同时期、不同用途的页面，统一存放便于管理和查阅。
- 本仓库不涉及任何特定网站或服务，仅作为静态页面的集合。

## 维护说明

- 欢迎提交 Pull Request 添加新的独立 HTML 页面，或对现有页面进行改进。
- 新页面请放置在 `pages/` 目录下，并尽量保持文件结构清晰。
- 公共资源建议统一放在 `assets/` 目录下，避免重复。
- 如有页面不再需要，可提交 PR 移除，并说明理由。
- 本仓库由维护者定期检查，确保页面可正常访问。
